🛠 校招雷达 · 三层爬虫工具策略

主选浏览器直连，两路本地 CLI 备用，覆盖反爬 / 验证码 / 全量库

① 主选 · Firecrawl

云端 REST API，浏览器直连，无需本地环境。适合 100 家公开官方招聘页批量抓取。免登录、自动处理反爬。

② 备用 · BrowserAct

本地 CLI browser-act，stealth-extract 零配置突破反爬、solve-captcha 自动解验证码。适合被拦截的站点，免费(登录)。

③ 备用 · OpenCLI

本地 CLI opencli + Chrome「Browser Bridge」插件，复用浏览器登录态，覆盖 51job 等需登录的全量岗位库。

用法：主选直接点「🕷 一键爬虫」即可联网抓取；两个备用工具需在本机终端跑 CLI 抓数据，导出 JSON 后点「📤 导入」即可进入雷达统一管理。

备用② BrowserAct（本地 CLI）

# 1. 安装 CLI（详见 https://github.com/browser-act/skills） npm install -g browser-act # 2. 登录（免费层需登录账号） browser-act login # 3. 零配置抓取受保护的招聘页 browser-act stealth-extract "https://join.qq.com/" --out ba_result.json # 4. 回到雷达页点「📤 导入」选择 ba_result.json

备用③ OpenCLI（本地 CLI + Chrome 插件）

# 1. 安装 npm install -g @jackwener/opencli # 2. 安装 Chrome「Browser Bridge」插件并完成登录 # 3. 抓取校招 / 小红书 / 微博招聘 opencli xiaozhao search opencli xiaohongshu search 校招 opencli weibo search 招聘 # 4. 导出 JSON 后在雷达页点「📤 导入」

说明：BrowserAct / OpenCLI 均为本地命令行工具，无法在纯前端网页内直接调用，故作为「备用」，通过「CLI 抓取 → 导出 → 导入」与雷达衔接。主选 Firecrawl 才是网页内一键直连的方案。
